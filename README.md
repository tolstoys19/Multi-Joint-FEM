# Multi-Joint-FEM

In here we provide a dataset with experimental and simulation results for a set of modular soft actuators attached to a :

Dummy Joint (DJ-SA Complex)
Multi-Joint Finger(FF-SA Complex)

Using different attachment methods:

Internal Attachment Method(IAM)
Chamber Constraint Attachment Method (CCAM)
External Elastic Band Attachment Method(EBAM)
Velcro Attachment Method (VAM)
Low Chamber Constraint Attachment Method (LCCAM)
Link Attachment Method (LAM)
Used for the article submitted to Robotics and Automation Letters titled: "Accounting for the Interaction between a Dummy Finger and Multi-Modular Soft Actuator for Multi-Joint Support using a Novel Multi-Joint FEM Dummy Finger-Soft Actuator Complex Model"

FEM Results for the DJ-SA Complex Model Are included in: FEM RESULTS DJ-SA COMPLEX.zip

FEM Results for the FF-SA Complex Model are included in: FEM RESULTS FF-SA COMPLEX.zip

EXPERIMENTAL Results for the DJ-SA Complex are included in: EXPERIMENTAL RESULTS DJ-SA COMPLEX.ZIP

EXPERIMENTAL Results for the FF-SA Complex are included in: EXPERIMENTAL RESULTS FF-SA COMPLEX.ZIP
